Présentation du ReadMe.txt

Aide pour avoir les images des skins, tags et les vidéos des émotes et répliques , ainsi que toutes les autres images
ou vidéo que ce soit pour la vidéo de présentation de la biopgraphie ou des compétences ou encore la miniature de la BD.

L'utilisateur doit aller dans son disque dur C , puis aller dans le dossier Utilisateur , puis à l'intérieur de ce dossier 
il doit créer le dossier hugod puis à l'intérieur de hugod, OneDrive , enfin le dossier Images
Le chemin doit correspondre à ce chemin :

C:\Users\hugod\OneDrive\Images

Ce qu'il faut mettre dans ce dossier Images c'est le dossier "Projet"  qui se trouve au niveau de Documentation et Programme dans l'arborescence de la forge.
